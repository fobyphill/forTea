Проект ForTea представляет собой учетно-историческую систему управления предприятием.
В данном репозитории лежит "пустая" система, т.е. без конфигурации.
Для запуска системы необходимо установить python и Django, затем запустить команду runserver.
При первом запуске система создаст недостающие данные в таблицах базы данных. В частности создаст пользователя с логином / паролем:  admin / admin.
Система использует админку Django, в которой можно изменить логин, пароль и другие данные пользователя, а также добавить других пользователей.
Инструкция по системе находится в файле manual.pdf
